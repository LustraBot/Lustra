Setup In Minutes

Invite Lustra using the button below. Choose the server where you want automated content delivery.
Confirm the bot can send messages, embed links, and use application commands in the channels you prefer.
Run /help for a guided walkthrough of every feature, or jump straight to /setup for a quick configuration.
Lean back and let Lustra handle the rest. Optional automation packs are available if you want scheduled drops.
